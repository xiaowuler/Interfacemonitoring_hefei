接口参数

ModeCode 必填 String 说明 要素产品模式名（例如：‘SPCC’，必须大写）
ElementCode 必填 String 说明 预报要素（例如：‘ER03’）
Latitude 必填 float 说明 纬度（29-35.1）
Longitude 必填 float 说明 经度（114.5-120）
ForecastLevel 必填 int 说明 预报层次
StartTime 必填 DateTime 说明 起始预报时间(yyyyMMddHHmm)
EndTime 必填 DateTime 说明 终止预报时间(yyyyMMddHHmm)
InitialTime 可不填 String 说明 起报时间(格式yyyyMMddHHmm)
